# discord-aoi-bot

Bu, aoi.js kullanılarak oluşturulmuş bir Discord bot şablonudur.

## Özellikler

- Komut yönetimi
- Olay yönetimi 
- Kolay yapılandırma

## Kurulum

1. Depoyu klonlayın:
   bash
   git clone <depo-url>

2. Proje dizinine gidin:
   cd discord-aoi-bot

3. Bağımlılıkları yükleyin:
   npm install

##  Yapılandırma

Bot tokeninizi ve komut önekini ayarlamak için config.json dosyasını düzenleyin.

## Kullanım

Botu başlatmak için:
node src/index.js
